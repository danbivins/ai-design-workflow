# Jira Ticket: Signup Landing Page

## Goal
Create a simple signup landing page for early access.

## User story
As a visitor, I want to enter my full name and email and submit the form so I can get early access.

## Requirements
- Heading with text "Sign up for the early access"
- Full name input
- Email input
- Submit button
- Error state for invalid email
- Success message after submission

## Constraints
- Desktop and mobile
- No backend (frontend-only)
