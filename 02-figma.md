# Figma Notes

## Design System
- Material Design 3 (Material You)
- Use Material 3 component conventions
- Do NOT use legacy Material Design (Material 2)

## Layout
- Single-column layout
- Centered content
- Max width ~760px

## Components Used

### Primary CTA Button
- Material 3 Filled Button
- Size: medium
- Shape: low corner radius (Material 3 default)
- No leading icon
- High contrast container and label colors
- Used as primary call to action

### Text Field — Email
- Material 3 Text Field
- Type: email
- Required
- No leading icon
- Supporting text below input
- Shows error state on invalid or empty submission

### Text Field — Full Name
- Material 3 Text Field
- Type: text
- Required
- No leading icon
- Supporting text below input
- Shows error state on empty submission

### Heading
- Material 3 typography
- Role: page headline
- Type scale: headline-large

## States
- Error state shown on text fields when submit is attempted with invalid or empty values
- Error message appears below the relevant input
- Focus states are visible for all interactive elements

## Visual Notes
- Button has strong visual contrast
- Layout prioritizes readability and clarity
- Clear separation between fields and actions

## Accessibility Intent
- All text readable at 200% zoom
- Fully keyboard navigable
- Clear, programmatic error messaging
